Fort Mapping Tool

In Sardegna, there are 26 fort locations (crossroads marked with numbers 1 through 26).

For the rules engine to know which fort gives influence to which province, we need a map showing what provinces each fort touches.

Click on a fort number on the map, then it will prompt you for the Province ID (1-16). Repeat this until all provinces touching the fort are added, then click cancel/blank to move to the next fort. Do this for all 26 forts, starting from 1 to 26.
